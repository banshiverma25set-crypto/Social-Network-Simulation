The aim of thi sproject is to develop an AI-based social network simulation system that models user interaction on social media platforms.

# AI Social Network Simulation

A C++ console-based social network simulation that uses Object-Oriented Programming concepts and basic AI techniques to analyze posts.

## Features
- User Registration System
- Verified Users
- Post Creation
- Emotion Detection
- Toxic Content Detection
- Trust Score Management
- Emotion-Based Feed
- Goal-Based Feed

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- STL (Vector, String)
- Basic AI Logic
- Git & GitHub

## OOP Concepts Implemented
- Classes and Objects
- Inheritance
- Encapsulation
- Polymorphism
- Dynamic Memory Allocation
